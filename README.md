This repository contains my implementations of fundamental Array problems in Python as part of my Data Structures and Algorithms practice.
The goal is to build strong problem-solving skills and prepare for coding interviews and aptitude tests.

All programs are implemented using basic Python constructs such as loops, conditions, and simple logic, focusing on understanding the core concepts rather than relying on built-in shortcuts.

🧠 Topics Covered

The following array problems have been implemented so far:

1️⃣ Sum of Elements in an Array

Calculates the total sum of all elements in an array using a loop.

Concepts Used

Iteration

Accumulator variable

2️⃣ Reverse an Array

Reverses the array using the two pointer approach.

Concepts Used

Two pointers

Swapping elements

3️⃣ Largest Element in an Array

Finds the maximum element by traversing the array once.

Concepts Used

Linear traversal

Comparison logic

4️⃣ Mean of Array

Computes the average value of elements.

Concepts Used

Sum of elements

Integer division

5️⃣ Second Largest Element in an Array

Finds the second largest number without sorting the array.

Concepts Used

Tracking two variables

Single pass optimization

6️⃣ Check if Array is Sorted

Checks whether the array is sorted in non-decreasing order.

Concepts Used

Adjacent element comparison

7️⃣ Remove Duplicates from Sorted Array

Removes duplicate values from a sorted array.

Concepts Used

Comparison with previous element

Building a result array

8️⃣ Maximum Consecutive 0s or 1s

Finds the longest sequence of consecutive identical bits.

Concepts Used

Counting consecutive elements

Updating maximum count

9️⃣ Smallest and Second Smallest Element

Finds the two smallest elements in the array.

Concepts Used

Tracking minimum values

Edge case handling

🔟 Array Subset Check

Checks whether one array is a subset of another.

Concepts Used

Sorting

Two pointer traversal
